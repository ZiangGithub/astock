# Astock

A股数据分析项目，基于akshare库进行股票数据获取和分析。

## 环境要求

- Python 3.8+
- akshare

## 安装

```bash
pip install akshare
```

## 项目结构

```
astock/
├── test_akshare.py    # akshare框架测试脚本
├── README.md          # 项目说明
└── .gitignore         # Git忽略文件
```

## 使用方法

运行测试脚本：

```bash
python test_akshare.py
```

## 功能

- 测试akshare库的基本功能
- 获取A股股票数据
- 股票数据分析和可视化
